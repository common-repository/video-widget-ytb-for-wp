Options for sharing video from YouTube.
Default size automatically according to the width of your web page.

== Installation ==

1. Upload the entire `youtube-video-custom` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Show in Appearance -> Widgets.
